 Este sistema de pedidos en línea para casas de comidas fue desarrollado originalmente para una pizzería y posteriormente mejorado para formar parte de mi portafolio de aplicaciones.
 
 Se puede probar ejecutando el servidor local con Django, previa activación del entorno virtual e instalación de sus dependencias especificadas en "requirements.txt".

 Accediendo al directorio raíz, se mostrará la página con la que interactúa el cliente para armar su carrito y confirmar su pedido. También podrá consultar el estado del mismo,
el cual será visualizado y gestionado por el administrador accediendo al directorio raíz seguido de "/gestionar", por ejemplo: "localhost:8000" y "localhost:8000/gestionar" respectivamente.

 La base de datos presente solo contiene Productos y Zonas ficticias para que se pueda probar el funcionamiento del sistema.
